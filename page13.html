Challenges Faced

  • 1. Integrating real-time chat and AI smoothly
    Ensuring that chat messages are transmitted instantly while the AI responds seamlessly, without lag or confusion, was a key challenge requiring WebSockets and async logic coordination.
  • 2. Managing async API calls
    Multiple asynchronous calls to the AI and backend had to be carefully managed to prevent race conditions, delays, or unexpected UI behavior.
  • 3. Handling invalid prompts
    Some user prompts were vague, offensive, or technically invalid. Creating a proper error handling and feedback mechanism for these cases was important for UX and AI performance.
  • 4. Making UI user-friendly
    Designing a layout that is clean, responsive, and intuitive required several iterations with feedback from users to optimize user engagement.
  • 5. Data storage structure in MongoDB
    Structuring messages, users, and AI response data in MongoDB collections in a scalable and efficient way was critical for future performance and analytics.
