Introduction

AICHATCODE is a modern real-time chat application inspired by the concept of WhatsApp group chats. It allows multiple users to join a group and chat with each other seamlessly. But what makes this project unique is its smart AI integration.

Whenever a user types @ai followed by any prompt in the chat, the message is processed, and a response is generated using Google Gemini AI. This means the chat app is not just for user-to-user conversations but also serves as an intelligent assistant within the group.

Problem Statement

  • Regular chat apps lack AI support
  • No easy way to ask AI questions within a group chat
  • Need for a system where humans + AI can chat together in real time
